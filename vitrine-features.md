# Optic Order - Frontend Features Overview
*Context for AI: Use this document to understand the user experience, interface features, and the direct value provided to the optical store staff when designing and writing copy for the showcase (vitrine) website.*

## 1. Executive Dashboard & Interactive Analytics
- **Utility**: A central command center for the optical store owner or manager.
- **What it provides**: A beautiful, real-time snapshot of business health. Features interactive revenue charts (daily/weekly/monthly), order status breakdowns via donut charts, and live KPI cards (revenue trends, active orders, total clients). It visually transforms raw data into instant business insights.

## 2. Intuitive CRM (Client & Prescription Management)
- **Utility**: Replaces paper files and messy spreadsheets with a blazing-fast digital interface.
- **What it provides**: 
  - Rapid search and filtering to find client records in milliseconds.
  - Custom-built, user-friendly forms to input complex optometric data (sphere, cylinder, axis, etc.) without confusion.
  - Seamless linking between a client and their historical prescriptions.

## 3. Streamlined Point-of-Sale & Order Creation
- **Utility**: The core daily tool for staff to process sales quickly and accurately.
- **What it provides**: A dynamic order creation interface that validates product stock in real-time. Staff can select a client, attach their specific prescription, pick products from a searchable catalog, and instantly calculate totals. It prevents human error by warning staff if a product is out of stock.

## 4. Professional Document Generation (Printables)
- **Utility**: Generates physical artifacts required for daily optical operations.
- **What it provides**: Instant, beautifully formatted, and printer-ready versions of Invoices (Bons de commande) and Medical Prescriptions. These include the store's custom logo, contact info, and perfectly aligned data tables, elevating the store's professional image to their customers.

## 5. Seamless Inventory Management UI
- **Utility**: Makes keeping track of frames, lenses, and accessories effortless.
- **What it provides**: Easily sortable and paginated tables displaying product categories, pricing, and current stock levels. Store managers can export their inventory to CSV with a single click for external auditing or accounting.

## 6. Role-Adaptive Interface
- **Utility**: Simplifies the interface for basic employees while giving full power to admins.
- **What it provides**: The UI automatically hides sensitive settings, financial reports, and user-management menus from regular staff. This guarantees a clean, distraction-free workspace for salespeople, while owners get the full suite of tools.

## 7. Modern, Responsive, and Premium Aesthetics
- **Utility**: Ensures the software is a joy to use on any device in the store.
- **What it provides**: Built with React, Tailwind CSS, and shadcn/ui, the platform feels lightning-fast, premium, and reliable. Smooth micro-animations, clear error/success notifications (toasts), and high accessibility standards reduce staff training time to near zero.
